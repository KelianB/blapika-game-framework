Class: ParticleSpawner

ParticleSpawner(config)

This class allows to spawn, render and update particles

Constructor

new ParticleSpawner(config)

Parameters:
Name Type Description
config Object The configuration of this particle spawner.
Properties
Name Type Attributes Default Description
spawnAngles Array.<Object> <optional>
 [{min: 0, max: Math.PI * 2}] An array of angles at which particles can spawn. Each angle can be either a number of an interval {min Number, max: Number}.
lifeTime Object | Integer <optional>
 -1 The amount of ticks after which a particle will be removed (either an int or {min: Number, max: Number}).
speed Object | Number <optional>
 5 The starting speed of particles (either a number or {min: Number, max: Number}).
spawnCount Object | Integer <optional>
 10 The amount of particles that will spawn by default when the spawn function is called (either a number or {min: Number, max: Number}).
position Object <optional>
 The default spawning position for particles. Structure: {x: Number | {min: Number, max: Number}, y: Number | {min: Number, max: Number}}.
Properties
Name Type Attributes Default Description
x Object | Number <optional>
 0 The x coordinate (either a number or {min: Number, max: Number}).
y Object | Number <optional>
 0 The y coordinate (either a number or {min: Number, max: Number}).
width Number <optional>
 32 The default width of particles.
height Number <optional>
 32 The default height of particles.
opacity Number <optional>
 1 The initial opacity of particles (between 0 and 1).
fadeInSpeed Number <optional>
 0 By how much the opacity of particles is incremented at each tick until reaching 1 (between 0 and 1).
fadeOutSpeed Number <optional>
 0 By how much the opacity of particles is decremented at each tick until reaching 0 (between 0 and 1). The fade-out starts before the particle's lifetime is reached, so that the opacity is at 0 right before the particle is removed.
customUpdate function <optional>
 A function that will be called on each particle update, with the particle as the this object.
customRender function <optional>
 A function that will be called after rendering the particle image and before removing the rotation, translation and opacity transforms. The particle is accessible as the this object.
customRenderAbsolute function <optional>
 A function that will be called after rendering the particle image and removing the rotation, translation and opacity transforms. The particle is accessible as the this object.
onParticleAdded function <optional>
 A function that will be called every time a particle is spawned. params: (particle, spawnIndex).
onParticleRemoved function <optional>
 A function that will be called every time a particle is removed. params: (particle).
image Image <optional>
 The image that will be used for rendering each particle. This property has priority over config.images.
images Array.<Image> <optional>
 An array from which the image of each spawned particle will be randomly selected.
xMovement Object <optional>
 An object that describes the horizontal movement of the particle.
